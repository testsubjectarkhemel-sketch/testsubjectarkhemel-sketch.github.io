ArkheTasks

BACKLOG → TODAY → DOING → DONE の 4 列で、やりたいことを整理する Android 向けタスク管理アプリです。

特徴

  • Backlog（やりたいことのストック）から Today/Doing/DONE へドラッグ感覚で移動
  • タスクタイトルとメモだけのシンプル設計
  • 完了履歴を「今週のDONE」として集計表示
  • データはすべて端末内に保存。外部サーバーは一切使用しません
  • エクスポート機能でテキストとしてバックアップ・共有可能（手動操作のみ）

プライバシー

ArkheTasks 本体はログイン機能もオンライン同期も持たず、タスク・メモ・完了履歴はすべて端末内のストレージにのみ保存されます。
詳細は プライバシーポリシー をご覧ください。
