Support ArkheTasks / Arkhemel

このページは、ArkheTasks や Arkhemel の活動を「ちょっと応援したい」と思ってくれた方向けのページです。

ゆるい応援のスタンス

ここでの応援は、生活に余裕があるときだけ・無理のない範囲で OK です。
「少しでも役に立った」「この世界線で活動を続けてほしい」と思ったときに、気が向いたら使ってください。

広告でゆるく応援

このボックス内には広告が表示されます（準備中）。
表示されるだけでも、わずかに収益につながることがあります。

ここに広告コードを入れる（AdSense 等）

投げ銭・寄付

もしアプリやコンテンツが役に立ったと感じたら、以下から任意で支援できます。

  • PayPal / クレジットカードで支援する
  • note / BOOTH などで支援する
  • Amazonほしい物リストから支援する
※ すべて完全に任意です。生活がまず最優先です。

広告付き動画を観て応援

下のボタンを押すと、広告が付いた動画が再生されます（YouTube など）。
再生してもらえると、少しだけ収益になります。

このページには第三者配信の広告が表示される場合があり、クッキー等を使用して閲覧情報が収集されることがあります。
詳細は各広告配信事業者のプライバシーポリシーをご確認ください。
ArkheTasks 本体のアプリは、タスク・メモ・完了履歴を端末内にのみ保存し、自動的な外部送信は行いません。
